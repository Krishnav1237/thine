# Thine: Demo Script (Current Platform)

Tight, punchy, and scroll‑stopping. Built to hook in under 1.5 seconds.

---

## Script (Scene by Scene)

### [0:00 – 0:03] Hook
- **Visual**: Black screen → instant text flash: “YOUR MEMORY IS NOT A SYSTEM.”
- **Audio**: Sharp click + bass hit.

### [0:03 – 0:07] Tension
- **Visual**: “MOST CONTEXT DIES IN A WEEK.”
- **Action**: Quick flashes of missed follow‑ups, messy notes, forgotten details.
- **Audio**: Low‑pass filter, micro‑drop.

### [0:07 – 0:12] The Challenge
- **Visual**: Phone snaps in. Quiz opens.
- **Text**: “TAKE 2 MINUTES.”
- **Audio**: Beat speeds up.

### [0:12 – 0:20] The Report
- **Visual**: The Intelligence Report animates in. Strengths/weaknesses, score band.
- **Text**: “GET A REAL SCORE.”
- **Audio**: Main drop.

### [0:20 – 0:28] The Arena
- **Visual**: Arena swipes (fast). Results pop after each take.
- **Text**: “HOW YOU ACTUALLY THINK.”
- **Audio**: Rhythm steadies, clean whoosh SFX.

### [0:28 – 0:35] The Loop
- **Visual**: Daily pack badge + streak chip.
- **Text**: “FRESH PACKS. DAILY.”
- **Audio**: Subtle rise.

### [0:35 – 0:45] The Save + Share Moment
- **Visual**: Scorecard share card → auth modal → leaderboard row.
- **Text**: “SAVE IT. SHARE IT. STACK XP.”
- **Audio**: Chime + clean click.

### [0:45 – END] Brand
- **Visual**: Logo + URL + “See how you actually work.”
- **Audio**: Final thud, then silence.

---

## Editing Rules
1. **Cut anything slow**: if a shot lasts > 2s, trim it.
2. **Prioritize pacing**: swipes, report reveal, and the save/share/leaderboard beat win.
3. **Subtitles always**: assume mute.
4. **Loopable ending**: end frame should match the start lighting.
