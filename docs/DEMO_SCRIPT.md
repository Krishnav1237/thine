## 🎬 Editing Rules (CRITICAL)
1. **CUT EVERYTHING SLOW**: If a shot lasts more than 2 seconds, trim it. 
2. **PRIORITIZE PACING**: Swipe motion, result reveal, and the challenge moment take priority over abstract UI pans.
3. **SUBTITLES ALWAYS**: Most people watch on mute. Use big, bold text.
4. **LOOPABLE START**: The end should feel like it can restart smoothly for infinite rewatches.
